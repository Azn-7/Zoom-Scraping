// Creating a .txt file to be parsed //

1. Create a .txt file
2. In the .txt file, each line must contain a link that directs to a google doc that you want parsed
3. Repeat step 2 for all links you want parsed
    3a The .txt should like this for example

        https://docs.google.com/document/d/15Es5UQ_0GtcoMp9-uGsTsXjWxf9Vd93HMiHy-eWRuys/edit?usp=drive_link
        https://docs.google.com/document/d/19Ya3J1DZ3SVinuPsnc1oVr11tU8AEyGOtekHV8DuPbs/edit?usp=drive_link
        https://docs.google.com/document/d/17QYt1-7ChCnb4irIn8MUGkdcISMWt_V1LXLwmKv-TAc/edit?usp=drive_link
        https://docs.google.com/document/d/1n6QlBmpHfuZ-Yt6sR3rOzTEj8y9Jf0dgD1fKXi2wzJc/edit?usp=drive_link
        https://docs.google.com/document/d/15l_St1Vb6wr5FJSz0qGDTcp1fWHYfK6DAy368u9nxHM/edit?usp=drive_link
        https://docs.google.com/document/d/12LTAyEky81CRH-3M1-9LzXsS6OV-ThjH6ofVr6wX9Sk/edit?usp=drive_link

4. Save the .txt file, name does not matter
5. Upload the .txt file to your google drive
6. Change sharable settings to public, and copy the link
7. In DocsToSheet.gs, on line 2, initialize publicDriveLink with the copied link
8. Change the sheetName the line below to whichever you would like to call it
9. Run :D
